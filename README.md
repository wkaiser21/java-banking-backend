# Java Banking Backend

A Java-based backend banking application. The project follows a **Test-Driven Development (TDD)** approach, emphasizing writing tests before implementing functionality and building reliable, maintainable code.

## Features

- Create and manage bank accounts
- Deposit and withdraw funds
- Transfer money between accounts
- Track account balances

## Technologies

- Java
- Gradle
- JUnit
- Git
- GitLab CI/CD (original project)
